Tento projekt je aplikácia na spracovanie textu

1. Požiadavky
Pred spustením aplikácie sa uistite, že máte nainštalovaný:

-Python 3.11
-Pip (správca balíčkov pre Python)

2. Inštalácia závislostí
Pred prvým spustením si nainštalujte všetky potrebné knižnice pomocou:

pip install -r requirements.txt

3. Spustenie aplikácie
Aplikáciu môžete spustiť nasledovne:

python main.py

Ak chcete použiť funkciu zarovnávania textu, môžete spustiť:

python align_text_window.py

4. Štruktúra projektu

📂 Projektový priečinok  
├── 📄 main.py                 # Hlavný skript aplikácie  
├── 📄 align_text_window.py     # Skript na zarovnávanie textu  
├── 📄 requirements.txt         # Zoznam závislostí  
├── 📄 README.txt               # Dokumentácia k projektu  

5. Ďalšie informácie
Ak sa vyskytne chyba pri importe knižníc, skúste spustiť projekt v virtuálnom prostredí.
